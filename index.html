MASTER

Gift Acknowledgement Standard Process

Identical across all gift types

This process is performed daily by the Legal / Philanthropy Strategy team, regardless of how the gift came in. The detail below expands the four condensed phases shown in every diagram above. The one variant — attaching a stock valuation sheet — applies only to securities gifts and is flagged accordingly.

PHASE 01
Generate
  • Log in to Bi-Launch Pad
  • Generate the batch of individual gift acknowledgement letters from the previous day
  • Generate organization / fund gift acknowledgement letters from the previous day
PHASE 02
Prepare
  • Open each letter in Adobe
  • Apply TCF letterhead and proper formatting
  • Fix donor address info in iPhi
  • Verify and correct salutations
  • Verify addresses
  • For Stock Contributions: locate the stock valuation sheet emailed by Accounting (Christine), save as PDF, remove unnecessary pages, attach to acknowledgement letter
PHASE 03
Produce
  • Print all the letters
  • Send digital PDF versions of letters to Nate
  • Save to OneDrive
  • Save into OnBase
  • Print addresses onto Avery label template
  • Prepare envelopes
PHASE 04
Distribute
  • Gifts for new funds or TCF employees: set aside for hand delivery
  • Review letters for accuracy
  • Assemble and insert letters into envelopes
  • Deliver to mailroom for mailing
